# Микросервисная платформа для интернет-магазина

## Описание
Платформа для автоматизации работы интернет-магазина. Состоит из двух микросервисов: 
- **Клиентский сервис** (`online-shop`) – публичный интернет-магазин для покупателей
- **Сервис сотрудников** (`workers`) – внутренняя система управления задачами для персонала

Сервисы независимы, взаимодействуют через REST API, что обеспечивает гибкость и масштабируемость.

## Функционал
### Интернет-магазин
- Регистрация и авторизация покупателей
- Просмотр каталога товаров с фильтрацией
- Управление корзиной покупок
- Оформление заказов и отслеживание статусов
- Личный кабинет пользователя

### Сервис для сотрудников
- Ролевая система (курьер, сборщик, кладовщик, менеджер, администратор)
- Постановка и распределение задач
- Отслеживание выполнения заказов
- Управление персоналом и правами доступа

## Технологический стек
- **Бэкенд:** Java 21, Spring Boot, Spring Data JPA, Spring Security
- **База данных:** PostgreSQL 
- **Миграции:** Flyway
- **Фронтенд:** Thymeleaf, HTML/CSS, JavaScript
- **Сборка:** Maven (многомодульный проект)
- **Архитектура:** Микросервисы, REST API, MVC

## Запуск проекта
1. Установите Java 21+, Maven, PostgreSQL
2. Создайте две БД в PostgreSQL (для каждого сервиса)
3. Настройте подключение в `application.yaml` каждого модуля
4. Выполните в корне проекта:
```bash
mvn clean install
```
5. Запустите сервисы:
```bash
# В первом терминале
cd online-shop
mvn spring-boot:run

# Во втором терминале
cd workers
mvn spring-boot:run
```
6. Откройте в браузере:
- Магазин: http://localhost:8080/register
- Сервис сотрудников: http://localhost:8082/register
